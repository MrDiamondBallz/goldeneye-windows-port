# Public / redistributable ROM fixtures

This directory is for ROM fixtures that are legal to redistribute in a public repository, such as:

- public-domain ROMs;
- homebrew ROMs with an explicit redistribution license;
- minimal test ROMs authored for this project;
- upstream test fixtures whose license allows redistribution.

Every committed ROM here must be accompanied by clear source and license notes.

Commercial copyrighted game ROMs do **not** belong here, even when a local user owns a legal personal copy. Keep those under ignored local-only paths such as `assets/` or outside the repo.
